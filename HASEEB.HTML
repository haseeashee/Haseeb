  • Module 3: JAVASCRIPT & ES6 FEATURES

  • Module 6: MONGODB DATABASE HANDLING

Syllabus

Prerequisite: Knowledge of basic programming, logic building, and internet usage.

Course Objectives:

  • Develop dynamic and responsive web applications
  • Understand full-stack development architecture
  • Deploy scalable and secure applications
Module Description Hours
1. INTRODUCTION TO WEB DEVELOPMENT History, Internet Concepts, Tools 4
2. HTML & CSS ADVANCED HTML5, CSS3, Flexbox, Grid 5
3. JAVASCRIPT & ES6 FEATURES Variables, Functions, Classes, Promises 5
4. CLIENT-SERVER COMMUNICATION AJAX, Fetch API, HTTP Methods 4
5. EXPRESS.JS BACKEND Server setup, Routing, Middleware 6
6. MONGODB DATABASE HANDLING CRUD Operations, Mongoose ODM 5
7. REACT.JS BASICS Components, Props, State, Hooks 6
8. DEPLOYMENT WITH HEROKU & NETLIFY Continuous Deployment, Hosting 5
9. INTRODUCTION TO CLOUD SERVICES (AWS) Hosting static websites, EC2 Basics 5
10. WEB APPLICATION SECURITY Authentication, HTTPS, Basic Encryption 5

Total: 50 Hours
